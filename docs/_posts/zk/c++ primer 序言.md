# c++ primer 序言
序言：c++11标准带来了巨大的进步，这些新的特性使c++变得更为完善
- 现代c++可以看做三个部分组成：
	- 低级语言，类似c
	- 现代高级语言特性
	- 标准库
- 结构：
	- 1与2：c++语言和标准库的基础内容。
	- 3与4：编写类的形式的抽象类型
