# vim中的buffer tab window区别
  
A buffer is the in-memory text of a file.  一个文件，可以在不同的窗口打开
  
A window is a viewport on a buffer.   类比win10中不同的窗口
  
A tab page is a collection of windows. 类比win10中的多个不同的桌面
