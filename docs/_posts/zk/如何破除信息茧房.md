# 如何破除信息茧房
- 人们虽然置身于海量信息中，但往往只会选择那些自己感兴趣的信息。长此以往，便会与其他领域的信息之间形成一道无形的墙，就像蚕宝宝一样被束缚在“茧房”之中。
- 在这样的“信息茧房”内，人们容易将自己的偏见当作真理，将他人的合理观点拒之于千里之外。
- **形成**：其概念与[证实性偏见](证实性偏见.md)相似，两者似乎从表现和概念都比较一致。
- **破除**：破除时由于[[逆火效应]]的原因，往往很难实现

---
需要以各种方式拓宽自己获取信息的渠道。
