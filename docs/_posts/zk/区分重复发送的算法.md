# 区分重复发送的算法

来源于计算机网络中的[RDT可靠数据传输](RDT可靠数据传输.md)的部分，通过对连续发送的分组进行0与1的编号，以区分两个连续的分组，以实现对重复的发送判断。很有启发性，实际上扩展开来，通过渐近式的方式以可靠数据传输方式的探究过程都是很有启发性的。
