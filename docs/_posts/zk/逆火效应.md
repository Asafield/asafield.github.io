# 逆火效应
心理学上的解释是「**当人们遇上与自身信念抵触的观点或证据时，除非它们足以完全摧毁原信念，否则会忽略或反驳它们，原信念反而更加强化**」。例：随着年龄的增长，我们就越固执己见，更难以接受新的事物。
